# SNav

저시력자를 위한 스마트 내비게이션

주요 특징:
- 사용자의 실시간 위치 정보를 확인해 음성으로 길 안내
- 큼지막한 글씨와 버튼과 고대비 색상을 사용한 저시력자를 위한 UI 사용
